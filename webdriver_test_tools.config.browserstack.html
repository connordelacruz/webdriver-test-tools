webdriver_test_tools.config.browserstack module¶

class webdriver_test_tools.config.browserstack.BrowserStackConfig[source]¶

Bases: object

Global configurations for BrowserStack testing

Variables:
  • BrowserStackConfig.ENABLE – Set to True to enable BrowserStack testing
  • BrowserStackConfig.USERNAME – Account username. Projects will need to override this
  • BrowserStackConfig.ACCESS_KEY – Access key. Projects will need to override this
  • BrowserStackConfig.BS_CAPABILITIES – BrowserStack test configurations Capabilities reference
  • BrowserStackConfig.BROWSER_TEST_CLASSES – Dictionary of browser test case classes to use indexed by the short name for that class (i.e. its command line argument).
