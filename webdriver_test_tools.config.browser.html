  • BrowserStackConfig.ACCESS_KEY – Access key. Projects will need to override this
  • BrowserStackConfig.BS_CAPABILITIES – BrowserStack test configurations. Capabilities reference
  • BrowserStackConfig.ENABLED_BROWSERS – Dictionary mapping browser names to a boolean. True enables the browser, False disables it. Defaults to Chrome and Firefox since they’re not platform specific
